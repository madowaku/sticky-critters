# Sticky Critters v0.2 Candidates (Backlog)

## 🔴 Must Fix (Important issues from beta)
- Performance optimization for > 300 notes.
- Better handling of multi-monitor DPI scaling.
- Clarify "Click-through" escape route for first-time users.

## 🟡 UX Polish (Minor refinements)
- **Group Title Toggle**: Show group header only on hover or hide it.
- **Improved Search**: Add tag filtering if tags are added.
- **Custom Fonts**: Allow choosing custom fonts for notes.
- **Dock Auto-hide**: Option to hide the AppDock when not in use.

## 🟢 Nice to Have (Small features)
- **Sketch PNG Export**: Export drawing notes as images.
- **Sticky Locking**: Option to "Pin" notes so they stay above everything else even in Board Mode.
- **Snooze Presets**: Custom snooze durations for Alarm Chicken.

## 🔵 Big Features (Next roadmap)
- **Sticky Links**: Connecting notes with dynamic lines (Arrows/Relationship lines).
- **OCR / Handwriting Recognition**: Basic text extraction from sketches or images.
- **Cloud Sync (Optional/External)**: Simple sync using existing cloud folders (Dropbox/OneDrive/etc.).
- **Rich Text / Markdown**: Basic markdown support within memo notes.
- **Nested Groups**: Allowing groups within groups for deep organization.

---
*Items will be prioritized based on feedback from v0.1 beta.*
