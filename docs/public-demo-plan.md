## Recording Shot List

Target format: 1080p, 30fps, 20-30 seconds. Record in a clean Windows desktop with a calm wallpaper. Keep the app window small enough that it feels like a resident desktop companion.

1. Open on an uncluttered desktop with Sticky Critters already visible.
2. Click `コード`, paste a short reusable snippet, and add it.
3. Click the copy action on the code note. Capture the copy feedback.
4. Add a short note with an alarm set a few seconds ahead. Let the alarm chicken toast appear.
5. Create or select two small notes and stash them into the hamster pouch.
6. Open the hamster pouch briefly, then close it.
7. End on a composed desktop with the code note, alarm note, pouch button, and app name visible.

## Capture Notes

- Avoid showing personal file paths, usernames, browser tabs, or private notes.
- Use sample code that is harmless, such as `npm run build` or a small CSS snippet.
- If notification permission prompts appear, grant permission before recording the final take.
- Keep the goat visible only as a background character unless the clip needs a delete action.
- Export as MP4/H.264. Keep an editable project file if using Clipchamp, DaVinci Resolve, or CapCut.
