# 🐾 Sticky Critters

**Sticky Critters** は、デスクトップをキャンバスに変える、キュートで高機能な付箋アプリケーションです。
Windows 環境に最適化され、Tauri + React + TypeScript で構築されています。

![Main UI Placeholder](docs/images/main_ui.png)

## 🌟 主な機能

- **多彩な付箋タイプ**: テキスト、コード、URL、ファイル、フォルダ、画像、そして自由な **スケッチ**。
- **直感的な整理**: 付箋を **グループ化** して一括移動や折りたたみが可能。
- **ほっぺ袋 (Stash)**: すぐには使わない付箋をハムスターのように「ほっぺ」に預けて、後で取り出せます。
- **ヤギのお腹 (Goat Belly)**: 削除した付箋はヤギが食べますが、お腹の中からいつでも救出可能です。
- **空間ナビゲーション**: **Sticky Map** (ミニマップ) で広大なワークスペースを俯瞰し、瞬時にジャンプ。
- **時間ナビゲーション**: 最近使った付箋や検索履歴から、過去の作業にすぐ戻れます。
- **アラーム鶏 (Alarm Chicken)**: 重要なタスクは鶏が時間にお知らせ。OS通知もサポート。
- **デスクトップ統合**: 透明なボードモード、クリック透過モード、常に最前面表示など。
- **トレイD&D**: ファイル / フォルダ / 画像をトレイへドロップして付箋化できます。複数まとめて落とすと **Bundle Note** になります。
- **小さい付箋表示**: 付箋はリサイズでき、設定から標準 / コンパクト / ミニ表示を選べます。

## 🛡️ 安全・安心設計

1. **実ファイルを削除しません**: ファイル/フォルダ付箋はショートカットとして動作します。アプリ内で削除しても元のファイルは安全です。
2. **コマンドを実行しません**: 安全のため、シェルコマンドの実行機能やプラグイン機能は排除されています。
3. **フォルダ内を勝手に列挙しません**: フォルダ付箋はパスを記録するだけで、中身を自動で読み込んだり監視したりしません。
4. **ローカル完結**: データはすべてあなたの PC 内 (`AppData`) に保存されます。勝手に外部サーバーへ送信することはありません。

## 🚀 開発環境のセットアップ

### 必要なもの
- Node.js (v18+)
- Rust (Tauri 開発用)
- Windows (推奨)

### インストール
```bash
npm install
```

### 開発実行 (ブラウザ版)
```bash
npm run dev
```

### 開発実行 (デスクトップ版)
```bash
npm run tauri:dev
```

### ビルド
```bash
# Webアセットのビルド
npm run build

# デスクトップ版のビルド (.exe)
npm run tauri:build
```
`npm run build` はブラウザ用の `dist` を作るだけです。トレイD&Dなどを含む Windows アプリ/インストーラーを更新するには、必ず `npm run tauri:build` を実行して新しい setup `.exe` を入れ直してください。
ビルドされたファイルは `src-tauri/target/release/bundle/` 内に生成されます。
デフォルトの配布物は NSIS setup `.exe` です。`.msi` はWindows標準のインストーラー管理に寄せたい場合に向いていますが、WiX/Windows Installer Service の状態に影響されやすいため、必要になった段階で MSI target を有効にしてください。

## 🐣 最初の使い方

1. アプリを開いたら、空のボード中央に表示される案内から **テキストを追加** を押します。
2. メモを書いて **追加** を押します。`Enter` で追加、`Shift+Enter` で改行できます。
3. **クリップボード** を押すと、コピー中のテキストをそのまま付箋にできます。
4. **スケッチ** を押すと手描き付箋を作れます。白いキャンバス上をドラッグすると描けます。
5. 画像ファイルをボードへドロップすると、画像プレビュー付きの付箋になります。
6. ファイル / フォルダ / 画像をトレイへドロップすると付箋化できます。複数まとめてドロップすると、1枚の **Bundle Note** に一覧化されます。
7. 付箋の右下をドラッグするとサイズ変更できます。**設定 > 外観 > 付箋サイズ** でミニ表示にもできます。
8. **設定** から常に最前面、デスクトップ表示、通知、自動起動、バックアップを調整できます。
9. **トレイへ** で隠した後は、通知領域のメニューまたは `Ctrl+Shift+Space` で戻せます。
10. クリック透過モードにした場合も、`Ctrl+Shift+Space` またはトレイメニューから編集モードへ戻せます。
11. **設定 > バックアップ > エクスポート** で付箋と設定を JSON として保存できます。

## 📁 データの場所
- **Windows**: `%APPDATA%/sticky-critters/`
  - `notes.json`: 付箋データ
  - グループデータは `notes.json` 内に保存されます
  - `settings.json`: アプリ設定
